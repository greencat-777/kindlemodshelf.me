Gnect

Gnect is a Kindle-friendly version of the classic Connect Four game. It includes a 7x6 board, multiple AI levels, and rendering optimized for e-ink displays.

Download

  • Download Gnect Extension (Latest Release)

Features

  • Classic 7x6 Connect Four board.
  • Adjustable AI difficulty levels.
  • E-ink friendly disc rendering (distinguishable by pattern and contrast).
  • Responsive touch controls.

Installation

  1. Download the kindle-gnect-extension.zip file.
  2. Unzip it and copy the resulting folder to the /extensions/ directory on your Kindle.
  3. Launch via KUAL -> Kindle Gnect -> Launch.
